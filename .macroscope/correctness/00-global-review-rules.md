Find is a local-first AI image intelligence app.

Review rules:
- Prioritize correctness, privacy, maintainability, and issue scope over style-only feedback.
- Do not suggest cloud uploads, hosted model APIs, telemetry, analytics, or third-party processing unless the linked issue explicitly asks for that architecture.
- Flag PRs that do not clearly solve the linked issue or that bundle unrelated refactors, formatting churn, cleanup, or opportunistic feature work.
- Treat user images, OCR text, captions, embeddings, face data, feedback data, and vault data as privacy-sensitive local data.
- Prefer practical, tested fixes over generic AI-generated patterns.
- Call out missing tests when behavior changes in upload, processing, gallery, search, clustering, people, vault, storage, worker jobs, or feedback flows.
